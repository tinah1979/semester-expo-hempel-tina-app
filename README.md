#RecipeBox -Tina Hempel

Recipe Box is a simple mobile app built with React Native and Expo Router that lets users save, view, and organize thier favoite recipes.

Features
add new recipes with Title, Category, Ingredients, Instructions, view full recipe details on a separate screen, clean, modern UI, Inspiration of the Day card with image, Persistent storage using AsyncStorage, Recipes Stay saved even after closing the app

App Screens 

Home Screen - Displays all saved recipes, Shows Inspiration of the Day, Button to add a new recipe

Add Recipe Screen
Form to enter recipe details, save recipe and returns to Home

Recipe Details Screen
shows full recipe information, back button to return to Home

Technologies used
React Native, Expo, Expo Router, AsyncStorage for persistence, TypeScript, React Hooks

How to run app

install dependencies - npm install

install AsyncStorage
npx expo install @react-native-async-storage/async-storage

Start the app  
npx expo start
